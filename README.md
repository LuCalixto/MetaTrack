MetaTrack 🚀

Sistema inteligente para gerenciamento e acompanhamento de metas pessoais, produtividade e evolução diária.

📌 Sobre o Projeto
O MetaTrack foi desenvolvido com o objetivo de ajudar usuários a organizarem suas metas, acompanharem seu progresso e manterem constância em suas rotinas de forma prática e intuitiva.
A proposta do projeto é unir produtividade, organização e uma experiência visual moderna em uma única plataforma.

✨ Funcionalidades
✅ Cadastro de metas
📈 Acompanhamento de progresso
🎯 Organização de objetivos
🔥 Interface moderna e responsiva
🌙 Dark Mode (em desenvolvimento)
📊 Dashboard de produtividade (em desenvolvimento)
🔐 Sistema de autenticação (em desenvolvimento)
